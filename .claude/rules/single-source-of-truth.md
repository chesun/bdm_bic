# Single Source of Truth: Paper-Centric Protocol

**The paper `paper/main.tex` is the authoritative source for all research content.** Everything else is derived.

## The SSOT Chain

```
paper/main.tex (SOURCE OF TRUTH)
  ├── talks/*.tex (derived — job market, seminar, short, lightning)
  ├── supplementary/*.tex (derived — online appendix)
  ├── figures/ (generated by analysis scripts, referenced by paper)
  ├── tables/ (generated by analysis scripts, referenced by paper)
  ├── references.bib (shared bibliography)
  └── replication/ (deposit package, assembled from scripts + data)

NEVER edit derived artifacts independently.
ALWAYS propagate changes from paper → talks.
```

---

## Talk Derivation Protocol

Talks are derived from the paper, not independently authored.

1. When paper content changes, flag affected talks for update.
2. Each talk format has different coverage:
   - **Job market** (40–50 slides): full results, mechanism, robustness highlights
   - **Seminar** (25–35 slides): main result, 1–2 robustness checks
   - **Short** (10–15 slides): key result + implication
   - **Lightning** (3–5 slides): hook, result, so-what
3. Figures and tables in talks must match paper versions exactly.
4. Notation in talks must match paper notation.

---

## Figure and Table Freshness

Before referencing any figure or table in a talk:

1. Check the analysis script's output is current (compare file timestamps — output newer than script).
2. Verify figure/table numbering matches the paper.
3. If the paper has been revised since the talk was last updated, flag for re-derivation.

---

## Content Fidelity Checklist (Paper → Talk)

- [ ] All results in talk appear in paper (no talk-only results)
- [ ] Effect sizes match exactly
- [ ] Standard errors / confidence intervals match
- [ ] Notation is consistent (same subscripts, same variable names)
- [ ] Citations in talk are a subset of paper citations
- [ ] Figures are identical (same script output, not redrawn)

---

## Enforcement

- `storyteller-critic` verifies paper-to-talk fidelity when dispatched on talks.
- `writer-critic` verifies internal paper consistency (tables vs prose) via the "tables are source of truth over prose" rule.
- Any deviation is a -10 deduction in talk scoring (see `quality.md` §3 Talks table).
